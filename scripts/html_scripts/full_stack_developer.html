  2. Job brief:

    We are looking for a Senior Web Developer to build and maintain functional web pages and applications. Senior Web Developer responsibilities include leading a team of junior developers, refining website specifications and resolving technical issues.
    To be successful in this role, you should have extensive experience building web pages from scratch and in-depth knowledge of at least one of the following programming languages: Javascript, Ruby or PHP.
    Ultimately, you will ensure our web pages are up and running and cover both internal and customer needs.


  3. Responsibilities:
    1. Identify user and system requirements for new websites and applications
    2. Prioritize software development projects, set timelines and assign tasks to team members
    3. Create wireframes to decide on layout
    4. Write or review code for various applications
    5. Ensure our software documentation is up-to-date
    6. Collaborate with mobile developers to build mobile-responsive websites

  4. Requirements:
    1. Work experience as a Senior Web Developer
    2. Expertise in at least one programming language, preferably Javascript, Ruby or PHP
    3. Solid knowledge of HTML/CSS
    4. Understanding of security practices
    5. Team management skills
    6. Familiarity with network diagnostics tools
    7. BSc/MSc in Computer Science or relevant field
